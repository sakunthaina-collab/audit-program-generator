## ⚡ Quick Start (3 Steps)

### **Step 1: Install Dependencies**

```bash
npm install
```

This will install:
- ✅ `react` + `react-dom` (v18)
- ✅ `lucide-react` (icons)
- ✅ `docx` (Word export)
- ✅ `file-saver` (download files)
- ✅ `tailwindcss` (styling)

---

### **Step 2: Add Component to Your Project**

```javascript
// App.jsx
import AuditProgramGenerator from './audit-program-generator-with-sql';

function App() {
  return <AuditProgramGenerator />;
}

export default App;
```

---

### **Step 3: Run**

```bash
npm run dev
```

Open browser: `http://localhost:5173`

**Done!** ✅

## 📝 Usage

### **1. Generate Audit Program**

1. Fill in the form:
   - What to audit (e.g., "Procurement process")
   - Industry (Banking, Retail, IT, etc.)
   - Audit type (Operational, Fraud, IT Security, etc.)
   - Known issues (optional)

2. Click **"สร้าง Audit Program"**

3. Wait 5-6 seconds (simulated AI generation)

---

### **2. Export to Word**

1. Review the generated audit program in tabs
2. Click **"Export Word"** button
3. Wait 2-3 seconds
4. File downloads automatically: `Audit_Program_[ProcessName].docx`

---

### **3. Export SQL Queries Only**

1. Go to **SQL Queries** tab
2. Click **"ดาวน์โหลด .sql"** button
3. File downloads: `audit_queries.sql`

## 🔧 Troubleshooting

### **Error: "Module not found: docx"**

```bash
npm install docx file-saver
```

### **Error: "Lucide icons not rendering"**

```bash
npm install lucide-react
```

### **Error: "Tailwind styles not working"**

Create `tailwind.config.js`:

```javascript
/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
```

Create `postcss.config.js`:

```javascript
export default {
  plugins: {
    tailwindcss: {},
    autoprefixer: {},
  },
}
```

Add to `src/index.css`:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

## 🎨 Customization

### **Change Colors:**

Edit in component:

```javascript
// Red flag highlighting
shading: { fill: 'DC2626' }

// Table headers
shading: { fill: 'E5E7EB' }

// SQL code blocks
shading: { fill: 'F3F4F6' }
```

### **Change Fonts:**

Add to Document properties:

```javascript
const doc = new Document({
  styles: {
    default: {
      document: {
        run: {
          font: 'Sarabun'  // Thai-friendly font
        }
      }
    }
  },
  sections: [...]
});
```

### **Add More SQL Queries:**

Edit `sqlQueries` object:

```javascript
const sqlQueries = {
  banking: [
    {
      id: 'my_custom_query',
      title: { th: 'Custom Query', en: 'Custom Query' },
      description: { th: 'Description', en: 'Description' },
      query: `SELECT * FROM table;`
    }
  ]
};
```
